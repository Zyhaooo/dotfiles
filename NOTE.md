# emacs

- 快速选择当前区域的内容 : `C-M-Space`
- 快速跳转到匹配的括号处(必须光标要在括号处,否则就是移动到上一个单词或下一个单词): `C-M-f \ C-M-b`
- 快速为选区两边添加括号(仅仅只是括号,别的不行): `M-(`
- 简单的替换: `M-%`
